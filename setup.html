Step 2 — How to Run on Any Computer
01

Download All The Files Above

Click each "Download" button above and save all files into the same folder on your computer. Example: C:\StreamGlide\

📁 Folder structure should look like: yt-dlp.exe, python-3.13.exe, app.py, requirements.txt, START_ENGINE.bat — all in the same place.
02

Install Python 3.13

Double-click python-3.13.exe to launch the installer. On the very first screen, check the box that says "Add Python to PATH" before clicking Install. This is critical — without it, nothing will work.

⚠️ If you forget to check "Add to PATH", uninstall Python via Control Panel and reinstall it with that option enabled.
03

Double-Click START_ENGINE.bat

Once Python is installed, go to your folder and double-click START_ENGINE.bat. A terminal (CMD) window will open. It will automatically:

1. Create a virtual environment (.venv)
2. Install Flask, yt-dlp and all packages
3. Start the backend server
4. Open http://localhost:5000 in your browser

04

Keep the CMD Window Open

The terminal (CMD/PowerShell) window that opened is the server engine. Do not close it while using the downloader. If you close it, the site will stop working.

✅ You will see "StreamGlide Engine is starting..." in the terminal when it is ready. The status badge at the top-right of the app will turn green and say SERVER ONLINE.
05

Start Downloading Videos!

Once the browser opens and the badge says SERVER ONLINE, paste any YouTube link below and click ANALYZE. Pick your quality and download!
